# Sistema de Biblioteca - Menú en C#

Proyecto académico desarrollado en C# como aplicación de consola.

## Descripción

Este proyecto implementa un sistema de gestión de biblioteca mediante un menú interactivo en consola.

El sistema permite navegar entre diferentes módulos:

- Gestión de libros
- Gestión de usuarios
- Gestión de préstamos
- Búsquedas y reportes
- Guardar / cargar datos

En esta primera fase del proyecto **solo se implementa la navegación del menú y las funciones que imprimen mensajes**, sin lógica real de base de datos o almacenamiento.

## Tecnologías utilizadas

- C#
- .NET Console
- Git
- Git Flow
